# dw-plugin-confightmlok

Enables raw HTML on Dokuwiki, like what 'config:htmlok' used to do in older DW versions.

All documentation for the BBCode Extensions Plugin is available online at:

  * https://www.dokuwiki.org/plugin:confightmlok

(Note, the page is named differently as the name 'plugin:htmlok' was already squatted)
